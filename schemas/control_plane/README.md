# Control-Plane Schemas

This directory holds schemas and model definitions owned by Actenon Cloud.

Current artifacts:

- `action-intent-submission-envelope.schema.json`
- `domain-model.yaml`
- `state-axes.yaml`

These define control-plane-owned envelopes, entities, and state models. They intentionally avoid redefining kernel-owned Action Intent, receipt, or verifier schemas.
