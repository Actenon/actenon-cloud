# Kernel-Aligned Schemas

This directory is reserved for pinned, imported, or referenced schemas that originate from the open execution kernel.

Current artifacts:

- `contract-boundary.yaml`
- `action_intent.finance.v1alpha1.schema.json`
- `receipt.finance.v1alpha1.schema.json`

At this stage the repository records the dependency posture only. Once the separate open kernel publishes versioned artifacts, this directory should hold pinned references or imported schemas without redefining their semantics locally.
